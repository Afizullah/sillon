# Vos dépendances ont aussi des licences

Que l'on parle d'outils tiers ou de librairies techniques, il est important de connaître leurs restrictions liées à l'usage afin de ne pas mettre votre produit dans une situation irrégulière.

Il faut par exemple, **à chaque fois que vous choisissez une librairie, vérifier sa licence** qui se situe normalement dans le repository du code source (ou adossée à la page de présentation de la forge).
