# Peer-programming & LiveShare

Quand on parle de communauté, il faut aussi penser "local", ça peut très bien être des développeurs environnants autour du projet. Il peut être intéressant d'avoir des moments de rencontre ou d'investigation afin de partager vos questionnements ou avancées.

Si jamais vous êtes à distance et que vous souhaitez vous plonger dans du code pour illustrer votre problème, sachez que [Visual Studio Code a un module LiveShare](https://marketplace.visualstudio.com/items?itemName=MS-vsliveshare.vsliveshare) afin que n'importe qui puisse ouvrir un éditeur virtuel pour interagir avec votre code local.
