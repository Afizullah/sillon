# DSFR, quesako ?

Le [SIG](https://www.gouvernement.fr/service-d-information-du-gouvernement-sig) (Système d'Information du Gouvernement) développe et maintient le "[Système de Design de l'État](https://www.systeme-de-design.gouv.fr)" (DSFR). Ce [design system](https://en.wikipedia.org/wiki/Design_system) permet aux produits numériques de l'État d'harmoniser leur apparence graphique tout en mutualisant l'effort en termes de développement graphique puisqu'il met à disposition un ensemble de composants prêts à l'emploi.

<CaptionedImage src={DsfrPreview} alt="" caption="Aperçu d'une page utilisant le DSFR" source="https://www.systeme-de-design.gouv.fr/a-propos" />

:::danger Utilisation
Le DSFR est un standard **obligatoire** dans l'administration, et son utilisation est **exclusivement réservée aux produits numériques de l'État**.
:::
