# Un support à disposition

L'équipe du DSFR met à disposition [un Slack](https://gouvfr.slack.com/) pour échanger voire pour demander de l'aide. Vous pouvez demander l'accès [ici](https://gouvfr.atlassian.net/servicedesk/customer/portal/1/group/1/create/9).

:::note
Il faut une adresse email terminant par `@*.gouv.fr` afin d'y être éligible. Si ce n'est pas votre cas et que vous travaillez sur le DSFR, vous pouvez toujours faire une demande. Dans le pire des cas et si vous êtes bloqués, vous pouvez toujours poster une issue [sur leur repository](https://github.com/GouvernementFR/dsfr).
:::
